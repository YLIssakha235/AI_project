# AI Project

## Description
This project focuses on two main artificial intelligence tasks:

- Academic performance prediction using Machine Learning
- Optical Character Recognition (OCR) for automatic exam correction

The project was developed as part of an engineering coursework project.

---

## Technologies Used

- Python
- Jupyter Notebook
- Scikit-learn
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib

---

## Machine Learning Models

### Academic Prediction
- Baseline Model
- Linear Regression
- Random Forest

### OCR
- Random Forest Classifier
- Multi-Layer Perceptron (MLP)
